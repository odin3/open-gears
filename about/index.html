Introduction

OpenGears it's free open-source PHP MVC framework. The main target of OpenGears it's to provide a lightweight basis for web applications. This framework is easy to use and can be useful for begginers who started to learn MVC pattern.

System requirements

To install and use OpenGears, you must have:
  • NGINX / Apache 2.2 or higher
  • Mod Rewrite for Apache
  • PHP 5.4 or higher
  • Installed PHP modules:
    • mb_string
    • MySQLi
  • MySQL Server 5.0 or higher (optional)

Features

  • Pure MVC structure
  • Fast setup and configuration
  • Easy template management
  • Build-in rounting
  • Fast Exception-based MySQL library
  • Output compression
  • Build-in cache system (in development)

File structure

Here is list of files and directories that included in the framework with their description:

  • application - application components
    • controller - application controllers
    • data - application misc data
    • model - models
    • views - views (templates)
  • media - images, styles and etc.
    • css - stylesheets
    • img - images
    • js - JavaScript files
  • system - system core components and drivers
    • drivers - system drivers (for example: MySQL)
    • controller.php - controller base class
    • kernel.php - Core
  • config.php - the framework configuration file
  • index.php - application entrypoint
